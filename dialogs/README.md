# Diálogos públicos

Os diálogos entre um ser humano e uma inteligência artificial
fazem parte integrante deste experimento.

Eles são publicados integralmente por meio de links públicos
gerados pela funcionalidade de compartilhamento do ChatGPT.

Cada link representa um recorte temporal do processo reflexivo.

Há dois tipos de diálogo, o diálogo de administração, planejamento e execução do experimento e o diálogo das perguntas eleitas.
